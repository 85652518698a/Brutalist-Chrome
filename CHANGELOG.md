## [1.0.0] — 2025-01-01

### Added
- Initial release
- Brutalist new tab page with animated background
- Live clock with timezone and 14-day calendar strip
- Real-time weather via Open-Meteo (no API key)
- Multi-engine search (Google, Bing, DuckDuckGo, Brave)
- Top sites shortcuts grid
- Pomodoro timer (25/5/15 min, configurable)
- Daily habit tracker with streak counter
- Persistent todo list with keyboard shortcuts
- Scratchpad notes (auto-save)
- Command palette (Ctrl+K)
- Rotating motivational quotes
- Ticker tape animation
- Parallax background on mouse move
- Glitch animation on brand hover
- Chrome Side Panel (todos, notes, history)
- Extension popup with quick stats
- Full settings/options page
- Context menus (Add to Todo, Notes, Search)
- Reading mode overlay (Alt+R on any page)
- Page scroll progress bar (all pages)
- Daily 8 AM motivational notification
- Data export as JSON
- Manifest V3 compliant
- Zero external dependencies (no npm for runtime)
